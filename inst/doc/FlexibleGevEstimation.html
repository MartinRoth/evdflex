Simple parameter model

fpar <- function(p, xpar) {
    loc   <- p[1]
    scale <- p[2]
    shape <- p[3]
    list(loc = loc, scale = scale, shape = shape)
  }

Fit for fixed 10 min data and for arbitrary 10 min data using the range likelihood.

fitSimple <- FitGevFlex(data[, m10], start = c(7, 2.5, 0.05), fpar = fpar)
fitRange  <- FitGevFlex(data[, .(m10, m20)], start = c(7, 2.5, 0.05),
                        fpar = fpar, likelihood = "range")

Plot of 10 min data (dots) and corresponding 20 min data (triangles) on a Gumbel scale. The standard fit is given in red and the range fit in blue.
